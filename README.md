# AI FAQ Chatbot

A modern AI-inspired FAQ chatbot built using HTML, CSS, and JavaScript.

## Features

- Interactive chat interface
- Quick suggestion chips
- Intelligent keyword matching
- Auto-scroll chat
- Programming and technology FAQs
- Mobile responsive design
- Clear chat command (`cls`)

## Technologies

- HTML5
- CSS3
- JavaScript

## Supported Topics

- React
- JavaScript
- Node.js
- Express.js
- MongoDB
- Git & GitHub
- Docker
- Kubernetes
- AI & Machine Learning
- HTML & CSS
- Python

## Usage

Type a question such as:

- What is React?
- Explain Docker
- What is AI?
- What is GitHub?
